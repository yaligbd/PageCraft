Deploy your Landing Page

From no account to a live site with your dashboard + Netlify.

Go Live in Minutes

1

Create your account

  1. Open Login and sign up (or sign in).
  2. You need an account to save pages and download the ZIP.
2

Build & Save your page

  1. Go to the Dashboard.
  2. Create or edit your landing page (Pages ‚Üí New Page).
  3. Click Save. Once saved, you can download a deployable ZIP.
3

Download your ZIP

  1. From your Dashboard, select the page.
  2. Click Download ZIP. The archive includes index.html and your assets.
Tip: Keep the folder structure intact when deploying.
4

Deploy to Netlify (recommended)

Option A ‚Äî Drag & Drop

  1. Open Netlify Drop.
  2. Drag your .zip file onto the page.
  3. Netlify gives an instant *.netlify.app URL.

Option B ‚Äî From Git

  1. Create a GitHub repo and upload your files.
  2. In Netlify: New site from Git ‚Üí choose GitHub ‚Üí select your repo.
  3. Static site: leave Build command empty, set Publish directory to repo root.
Custom domain: Site ‚Üí Domain settings ‚Üí Add custom domain and follow DNS steps.
5

Alternative free host ‚Äî GitHub Pages

  1. Create a free GitHub account.
  2. Create a public repository (e.g., my-landing).
  3. Unzip your download and upload contents so index.html is at repo root.
  4. Repo ‚Üí Settings ‚Üí Pages ‚Üí Branch: main, Folder: /(root) ‚Üí Save.
  5. After ~1 minute: https://<user>.github.io/<repo>/ is live.
If hosted under a subpath, use relative links like ./about.html (not /about.html).
